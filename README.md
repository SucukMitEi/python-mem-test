# Foobar

Foobar is a Python library for dealing with word pluralization.

## Installation instructions

1.
2.

## Usage

```python
from mem import MEM

# open a process
mem = MEM(12345) # by PID
# or
mem = MEM.fromName("konsole") # by name

value = mem.readInt(0x7fffffff)
print(value)
```
